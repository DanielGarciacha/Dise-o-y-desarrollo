Actividades Deportivas y Recreativas

En esta página podrás encontrar información sobre los diferentes deportes y actividades recreativas que ofrece nuestra universidad. Conoce los requisitos necesarios para participar en cada una de estas actividades y únete a nuestra comunidad estudiantil.

Fútbol

Únete a nuestro equipo de fútbol y participa en los torneos interuniversitarios. Entrenamientos tres veces por semana.

Taekwondo

Aprende artes marciales, mejora tu disciplina y condición física con nuestros instructores certificados.

Baloncesto

Forma parte de nuestro equipo de baloncesto y compite en los torneos regionales universitarios.

Danza

Expresa tu arte a través del movimiento. Ofrecemos clases de diferentes estilos de danza.

Música

Aprende a tocar un instrumento o forma parte de nuestra banda universitaria.

Natación

Clases para todos los niveles, desde principiantes hasta competencia.

Requisitos para Fútbol
  • Nombre completo (según documento de identidad)
  • Número de documento (C.C. o T.I.)
  • Programa académico que estás cursando
  • Cuatrimestre actual
  • Certificado médico de aptitud física (vigente)
  • Seguro estudiantil activo
Requisitos para Taekwondo
  • Nombre completo (según documento de identidad)
  • Número de documento (C.C. o T.I.)
  • Programa académico que estás cursando
  • Cuatrimestre actual
  • Certificado médico de aptitud física (vigente)
  • Seguro estudiantil activo
  • Uniforme blanco (dobok) para prácticas
Requisitos para Baloncesto
  • Nombre completo (según documento de identidad)
  • Número de documento (C.C. o T.I.)
  • Programa académico que estás cursando
  • Cuatrimestre actual
  • Certificado médico de aptitud física (vigente)
  • Seguro estudiantil activo
  • Tenis deportivos adecuados para cancha
Requisitos para Danza
  • Nombre completo (según documento de identidad)
  • Número de documento (C.C. o T.I.)
  • Programa académico que estás cursando
  • Cuatrimestre actual
  • Ropa adecuada para prácticas (leotardos, mallas, etc.)
  • Zapatos de danza según el estilo elegido
Requisitos para Música
  • Nombre completo (según documento de identidad)
  • Número de documento (C.C. o T.I.)
  • Programa académico que estás cursando
  • Cuatrimestre actual
  • Instrumento propio (opcional, la universidad provee algunos)
  • Conocimientos previos (si aplica)
Requisitos para Natación
  • Nombre completo (según documento de identidad)
  • Número de documento (C.C. o T.I.)
  • Programa académico que estás cursando
  • Cuatrimestre actual
  • Certificado médico de aptitud física (vigente)
  • Traje de baño adecuado
  • Gorra y gafas de natación (recomendado)
